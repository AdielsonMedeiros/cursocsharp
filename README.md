 Repositório de Estudos

Este repositório contém a organização de exercícios, métodos e desafios realizados durante o aprendizado da linguagem C# e plataforma .NET.

## Estrutura do Projeto

O projeto está organizado em subdiretórios categorizados por tipo de atividade:

### Desafios Concluídos
Nesta seção estão os projetos que exigiram a aplicação de múltiplos conceitos para resolver problemas específicos (localizados na pasta `Desafios/`):

*   **CalculoRota**: Sistema para cálculo de rotas, consumo de combustível e custos de viagem.
*   **Exames**: Sistema para processamento e validação de exames laboratoriais (Glicemia, Colesterol, Triglicerídeos).
*   **SistemaFinanceiro**: Projeto estruturado focado em gestão financeira e conceitos de domínio.

### Exercícios e Fundamentos
Projetos voltados para a prática de sintaxe básica e fundamentos da linguagem:

*   **Exercicios**: Coleção de práticas pontuais de lógica e sintaxe.
*   **Metodos**: Exploração de criação de métodos, classes e manipulação de objetos.

## Como Executar

O repositório utiliza o formato de solução unificada do .NET. Para abrir todos os projetos simultaneamente, utilize o arquivo:

*   `Curso.slnx`

Você também pode navegar em pastas específicas e executar o comando `dotnet run` dentro de cada diretório de projeto.
